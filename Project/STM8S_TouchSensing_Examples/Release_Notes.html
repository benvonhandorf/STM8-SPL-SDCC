Update History

  

V2.5.0 / 14-October-2013

 

Main Changes

•  Suppression of STM8L devices examples.

   Note: The STM8L examples are now in the new STM8L STMTouch Library.

•  Remaining STM8S examples have been tested with the "STM8S TouchSensing Driver V2.5.0".

 

Content

•  STM8S207K6_Ex01: 5 touchkey sensors.

•  STM8S207K6_Ex02: 1 linear sensor (slider).

 

Development Toolchains and Compilers

•  ST Visual Develop (STVD) v4.3.4.

•  Cosmic STM8 C Compiler v4.3.9.

•  Raisonance Ride7/RKit-STM8 v2.51.13.

•  IAR Embedded Workbench for STM8 v1.41.

 

Supported Devices and EVAL boards

•  STM8/128-EV/TS board (composed of MB631 board and STM8SxxxK-TS1 daughter board).

 

Known Limitations

•  None.

  

V2.4.0 / 01-July-2011

 

Main Changes

•  All examples tested with STM8 Touch-Sensing Driver v2.4.0.

•  New examples added for STM8L15x LowDensity devices with Charge-Transfer acquisition mode.

•  Examples folder name changed.

  

V2.3.1 / 14-February-2011

 

Main Changes

•  All examples tested with STM8 Touch-Sensing Driver v2.3.1.

 

Content

•  STM8L10x_CT_Example_FW:

º  Firmware with single-channel keys only using Charge-Transfer acquisition.

•  STM8L10x_RC_Example_FW:

º  Firmware with single-channel keys only using RC acquisition.

•  STM8L15x_CT_Example_FW:

º  Firmware with single-channel keys only using Charge-Transfer acquisition.

º  To be used with the STMT/8L-EV1 board.

•  STM8L15x_CT_WheelExample_FW:

º  Firmware with one wheel only using Charge-Transfer acquisition.

º  To be used with the STMT/8L-EV1 board.

•  STM8S207_RC_Example_FW:

º  Firmware with single-channel keys only using RC acquisition.

º  To be used with the STM8/128-EV/TS board (composed of MB631 board and STM8SxxxK-TS1 daughter board).
